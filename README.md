Theme Changer 🌙☀️

A simple, clean light/dark theme toggle for your web projects. Click the button, switch themes, and it remembers your preference.

Features


Toggle between light and dark modes — smooth transitions
Remember user choice — uses localStorage to save preference
Visual feedback — sun icon for light mode, moon icon for dark mode
Minimal setup — just HTML, CSS, and JavaScript


How It Works


Click the button → theme switches
Colors update → background, text, button all change
Preference saved → localStorage remembers what you picked
Next visit → loads your saved theme automatically


Files


theme.html — HTML markup with button and icons
style.css — styling for button and theme colors
theme.js — JavaScript logic for toggle and localStorage


Usage


Copy the HTML structure into your project
Link the CSS file
Link the JavaScript file
Done — button is ready to use
